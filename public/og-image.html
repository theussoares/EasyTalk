Como gerar a imagem Open Graph:

  1. Clique no botão "Gerar Imagem PNG" abaixo
  2. A imagem será baixada automaticamente
  3. Renomeie o arquivo para "og-image.png"
  4. Coloque o arquivo na pasta public/ do projeto
  5. Atualize as meta tags para usar "/og-image.png"
